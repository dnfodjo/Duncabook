Homepage 1513 Installation Instructions

Step 1: Upload to Server

Copy the contents of the "upload" folder to your store's top level directory, preserving the directory structure. This will not overwrite anything.

Step 2: Install Module

Log in to your store's administration and, from the menu, select "Extensions->Modules", then click the "Install" link in the "Action" column for the "Homepage" module. You may then edit the module to enable it and set its sort order.

Clicking the "Edit" link for the "Homepage" module will automagically insert the following tables into your database:–

homepage
homepage_description
homepage_to_store

The new tables will be prefixed with your database prefix.

Step 3: Stylesheet

If you have Qphoria's VirtualQMod system installed, the following is not necessary as the included vQmod file will do it for you.

Add the following to the head section of "catalog/view/theme/default/template/common/header.tpl":–

<link rel="stylesheet" type="text/css" href="catalog/view/theme/default/stylesheet/homepage.css" />

That finishes the installation.

Adding Entries to Your Home Page

Login to your OpenCart Admin. From the menu, select "Extensions–>Modules", then click the "Edit" link for the "Homepage" module.

Clicking the "Homepage Entries" button will take you to the list page where you may add/edit your homepage entries. If you are using Qphoria's VirtualQMod system, you will also find an entry under the "Catalog" menu, titled "Homepage Entries".

By clicking the "Data" tab you will be able to set the status of the entry to "Enabled" or "Disabled" (disabled by default). You can also set the sort order on this page.

When done, click "Save". This entry will now appear on your home page (if enabled).

Each homepage item will appear on your home page with the title as the heading and the description as the subject matter below the heading.
